## Getting Started

### Prerequisites

```bash
node -v >= 18
yarn -v >= 1.22
```

### Installation

```bash
yarn install
```

### To Start Development Server

```bash
yarn dev
```

### To Run Test Suite

```bash
yarn ci:unit        // Unit Test
yarn ci:e2e         // e2e Test
yarn ci:all         // Unit and e2e Test
```

### To Build

```bash
yarn build
```

### To Build and Start

```bash
yarn start:static
```

Visit [http://localhost:3000](http://localhost:3000)
